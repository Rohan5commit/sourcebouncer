# Demo Script (2-5 minutes)

## Narration

**[0:00-0:30] Introduction**
"SourceBouncer is a paid verification agent for the CROO agent economy. Other agents hire it through CAP to verify claims and get trust-scored evidence packages."

**[0:30-1:00] The Problem**
"AI agents are multiplying fast. Bad outputs destroy trust. Most agents cannot buy verification from another agent. SourceBouncer solves this."

**[1:00-2:00] Live Demo**
1. Open the Demo page
2. Enter 3 claims (one true, one false, one disputed)
3. Add source materials
4. Select Deep Verification tier ($2.00 USDC)
5. Submit and show the verification running
6. Show the Trust Report with claim-by-claim verdicts

**[2:00-3:00] CAP Flow**
1. Show the Audit view with CAP invocation
2. Highlight the escrow and settlement records
3. Show the on-chain transaction hash

**[3:00-4:00] Agent Reuse**
1. Navigate to the Reuse page
2. Show how downstream agents consume the output
3. Demonstrate ResearchBrief Agent revising content

**[4:00-4:30] Ask SourceBouncer**
1. Ask why a claim was marked weak
2. Show grounded Q&A from verification data

**[4:30-5:00] Summary**
"SourceBouncer proves paid verification is a commercially viable agent dependency. Listable on CROO Agent Store. Callable by any CAP agent. Settles USDC on-chain."
